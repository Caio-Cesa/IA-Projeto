# IA Projeto - Dashboard de Pareceres

Este repositório contém a proposta de acompanhamento estratégico (Dashboard) do projeto de Inteligência Artificial focado em Pareceres para a Lei de Licitações.

> ⚠️ **Status do Repositório/Dashboard:**
> Este documento visual e a métrica de dados apresentada são atualmente uma **proposta em andamento**, aguardando validação e aprovação da equipe para ser o método oficial de verificação de status do projeto.

## Estrutura do Dashboard

- `index.html`: Interface visual que exibe as métricas e o progresso do projeto.
- `style.css`: Estilização corporativa e responsiva do dashboard.
- `data.json`: Arquivo de dados concentrando os marcos, métricas de precisão, histórico de pivotagens (experimentos) e status geral. O consumo e edição deste arquivo formam a base do painel interativo.
